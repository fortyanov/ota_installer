=======Инсталлятор ОТА============

ПРИМЕР:
python ota_install.py --user fedor --host 172.20.3.17 --name ota-keln --path /opt/
потом при установке указываете пароль

ПРИМЕЧАНИЕ:
для корректной установки необходим доступ в интернет
ota_install.py, ota_install_stuff и папка с приложением (ota_web) должна быть расположены в одной дирректории
